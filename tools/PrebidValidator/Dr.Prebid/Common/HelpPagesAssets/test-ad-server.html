Ad Server Setup Validation

Overview

This test determines whether your Prebid line items are configured correctly in your primary ad server.

  • Ad server request sent with key-value targeting
    Verifies whether the bid request was sent. The request will not be sent if the IDs you entered on the Setup page are incorrect. Tap this entry to see a list of key-value pairs that were sent with this request.
    • Select Prebid Key-Value Pairs to view only the key-values sent that apply to prebid.
    • Select Ad Server Request to view the full list of key-values sent in the ad request.
  • Prebid Mobile creative HTML served
    Tap this entry to view:
    • the Prebid.js HTML code for the Prebid Mobile creative.
    • the creatives that were returned. In a successful test two creatives will be returned: the creative returned from the ad server and the Prebid.org test creative. Tap the buttons to view each creative.

Results

Success: If the test is successful, all entries will display green check marks. Tap into the Prebid Mobile creative HTML served screen to compare the ad server creative to the Prebid.org test creative. The Received Creative and the Expected Creative should be identical.

Failure: If the test fails, check the key-value targeting to ensure the bid price and size match those you've defined in your Prebid line item ad server settings. Also ensure that the creative HTML is the expected HTML.
